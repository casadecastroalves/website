# Guia — Etapa 1 · Identidade e Memória

Mapa territorial da Rede **Movimento Irun** para oficinas comunitárias.

## Como abrir

1. Abra **`identidade-e-memoria.html`** no navegador (Chrome, Firefox ou Edge).
2. O mapa abre com os **27 Territórios de Identidade da Bahia** (polígonos brancos).
3. Internet é necessária para o mapa base, fotos e vídeos. Textos e polígonos funcionam parcialmente offline.

Atalho: `mapa.html` redireciona para o mapa principal.

### URLs úteis

| Visualização | URL |
|--------------|-----|
| Bahia (27 TI) | `identidade-e-memoria.html?vista=ti` |
| Rede (6 territórios) | `identidade-e-memoria.html?vista=rede` |
| Quilombo Lagoa Grande | `identidade-e-memoria.html?territorio=lagoa-grande` |
| Itaparica + RM | `identidade-e-memoria.html?territorio=itaparica-rm` |

Use **Compartilhar** para copiar o link ou **Incorporar** (iframe) da visualização atual.

## Formulário (oficina)

Abra **`formulario.html`** para registrar pontos no mapa. Gera JSON para copiar — coordenadas podem ser coladas do Google Maps / My Maps.

## Arquivos

| Arquivo | Função |
|---------|--------|
| `identidade-e-memoria.html` | Mapa principal |
| `dados.js` | Conteúdos da Rede (editar na oficina) |
| `ti_bahia_geo.js` | 27 polígonos TI |
| `lagoa_grande_geo.js` | 7 subregiões Lagoa Grande |
| `roteiro_contra_costa_snippet.js` | Roteiro Itaparica |
| `imagens/lagoa-grande/` | Fotos piloto |
| `formulario.html` | Registro de pontos |

## Menu lateral (Rede)

- **Território** — texto do território
- **Cultura** — narrativas (com “Ver no mapa” quando aplicável)
- **Galeria** — fotos e vídeos
- **Memória** — blocos de memória coletiva
- **Roteiros** — percursos com linha no mapa

## Camadas no mapa (chips)

Ambiente · História · Instituições · Produtos

## Piloto de conteúdo

**Quilombo Lagoa Grande:** textos, 22 fotos, 3 vídeos YouTube, AQCOMAQ, timeline 2013/2016/2022.

**Itaparica:** roteiro Contra Costa (pontos + linha), projetos da Rede.
